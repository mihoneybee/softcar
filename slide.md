# Slide 3: Tags Principais - HTML (Estruturação)
* `<div>`: Blocos fundamentais para containers e estruturação do menu.
* `<h1>` a `<h3>`: Hierarquia semântica para títulos e perguntas do briefing.
* `<a>`: Navegação interna entre páginas e links externos de referência.
* `<hr>`: Divisores visuais que reforçam a organização clássica e limpa.
* `<img>`: Renderização das vitrines dos veículos com foco em performance visual.

---

# Slide 4: Tags Principais - CSS (Estilização)
* `position: sticky`: Mantém o menu fixo no topo, otimizando a UX.
* `backdrop-filter: blur()`: Efeito *Glassmorphism* (vidro fosco) para modernidade.
* `display: grid`: Layout responsivo que adapta a vitrine de 3 para 1 coluna.
* `transition`: Animações suaves de *hover* em botões e cards de carros.
* `radial-gradient`: Iluminação de fundo que destaca os modelos de luxo.
